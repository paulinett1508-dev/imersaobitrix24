# Imersão Bitrix24 — Landing Page

Landing page de alta conversão para a **Imersão Bitrix24** da Traevo.  
Evento semanal ao vivo, toda quarta-feira às 11h. Gratuito.

## Stack

- **Vite** — build tool
- **Tailwind CSS v4** — utilitários e design tokens
- **JavaScript puro** — sem frameworks

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy

Configurado para Vercel (`vercel.json`). Output em `dist/`.

## Assets

Coloque os arquivos de imagem nas pastas abaixo antes de referenciar no HTML:

| Pasta | Conteúdo |
|---|---|
| `src/assets/images/logo/` | Logotipos da Traevo |
| `src/assets/images/apresentadora/` | Foto da apresentadora |
| `src/assets/images/depoimentos/` | Fotos dos depoimentos |
| `src/assets/images/og/` | Imagem Open Graph (1200×630px) |
| `src/assets/icons/` | Ícones SVG avulsos |
